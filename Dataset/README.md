
This folder contains the dataset files used to train and evaluate the classification models:

Source: Kaggle - Mental Health Text Classification Dataset 

Dataset - https://www.kaggle.com/datasets/priyangshumukherjee/mental-health-text-classification-dataset

Features: 21 engineered features (including TF-IDF vectors, sentiment polarity, subjectivity, and linguistic ratios).

Classes: 4 labels (Anxiety, Depression, Suicidal, and Normal).

Participants: Thousands of unique social media contributors (Reddit/Twitter).

Test CSV: mental_health_combined_test.csv

Train CSV: mental_heath_unbanlanced.csv
